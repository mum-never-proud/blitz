How to use blitz

From my personal experience i can say that there are times where we end up using heavy frameworks to meet our basic needs.

Blitz intended for DOM Mainpulation and sticks to it!

Getting started

Selector

From simple to complex queries, everything is being taken care of!

            
              $('.blitz');
              $('#blitz');
              $('div > span');
              $('li:nth-of-type(even)');

Supports Chaining

Chain it! Until you find the deadend.

            
              $('.blitz').css('color', 'red').on('click', () => {});

CSS Mainpulation

CSS Styling for the elements (applies to all selected elements)

            
              $('.blitz').css('color', 'red'); // sets color to red
              $('.blitz').css('color'); // gets element color
              $('.blitz').css({ background: 'red', color: 'white' }); // sets multiple css properties

Class Manipulation

  • Adds, Removes or Toggles a class
  • Check for the existance of a class
            
              $('.blitz').addClass('blitz-class'); // adds a class
              $('.blitz').removeClass('blitz-class'); // removes a class
              $('.blitz').toggleClass('blitz-class'); // toggles a class
              $('.blitz').hasClass('blitz-class'); // checks if 'blitz-class' exists

DOM Manipulation

  • Adds, Removes, Appends, Prepends an element
  • Gets children of an element
  • Checks if the given element is inside the target element
  • Gets value or inner HTML of an element
            
              $('.blitz').add(givenElement); // adds an element
              $('.blitz').remove(); // removes all elements
              $('.blitz').prepend(givenElement); // prepends an element
              $('.blitz').append(givenElement); // appends an element
              $('.blitz').before(givenElement); // inserts before the target element
              $('.blitz').after(givenElement); // inserts after the target element
              $('.blitz').children(); // return children of the first element
              $('.blitz').eq(0); // slices off the first element of the selected elements
              $('.blitz').contains(givenElement); // checks if the target element contains the given element

Events

Bind or Remove events without hassle of storing the reference of the event callback function

            
              $('.blitz').on('click', () => {}); // binds click listener to the elements
              $('.blitz').off('click'); // removes click listener from the elements
              $('.blitz').off(); // removes all events listeners from the elements
              $('.blitz').off('click', () => {}); // removes specified callback of click event

Yay Blitz!

Ease your DOM Manipulation with Blitz & Happy coding

            
              $('.blitz')
                .on('click', () => {})
                .removeClass('blitz-class')
                .addClass('another-blitz-class')
                .css('color', 'red')
                .off();
